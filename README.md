## 先说 DMIT 是谁

如果你刚接触 DMIT 这个名字，花两分钟搞清楚背景很有必要。

DMIT（dmit.io）是 2017 年前后开始跑 VPS 业务的主机商，美国纽约注册，华人团队运营，目前主营三个数据中心：美国洛杉矶（LAX）、中国香港（HKG）、日本东京（TYO）。

## 它适合谁，不适合谁

直接分场景说：

**适合：**
- 搭梯子 / 科学上网，需要稳定节点，电信用户选 CN2 GIA 没有理由后悔
- 个人博客、WordPress 小站，1核1G 对静态网站完全够
- 流媒体解锁，DMIT 全系配原生 IP，实测能解锁 Netflix、TikTok 等主流平台（官方不保证，以实测为准）
- 外贸 / 跨境轻量需求，收发邮件、访问 Google Workspace，跑得很顺

**不适合：**
- 跑数据库密集型应用（MySQL、Redis），内存不够
- 高并发 Web 服务，1 vCPU 会是瓶颈
- 需要稳定大流量跑数据，1TB/月上限不是很宽裕，得看你实际使用量

说句题外话，如果你的需求是"有一台靠谱的美区节点、随时能用、不用操心晚高峰掉速"，Malibu 在这个价位里没有真正的对手。

## 新手注意：这几点很多人不知道

买之前把这几条过一遍，能省不少麻烦：

1. **注册后等 7 天才能下单。** 这是 DMIT 对新账号的验证机制，不是 BUG，也不是针对所有人，但如果你是刚注册，记得提前几天把账号搞定，别等到看见补货了再注册。

2. **SSH 默认是密钥登录，没有密码。** 机器到手之后，在控制台找 Access 下载密钥压缩包，里面的 `.ppk` 文件配合 PuTTY 或者直接用 SSH 客户端导入就能连。不习惯密钥的可以连上去之后再改成密码登录。

3. **IP 被墙可以免费换，但有条件。** Premium 和 Eyeball 系列免费换 IP，前提是 IP 真的被 GFW 封了，每 15 天可以免费换一次，其他情况换一次 $5。比搬瓦工要实在一些，至少不用等。

4. **退款政策是真实的。** 3 天内流量用量不超过 30GB，可以全额退款（扣支付网关手续费）。30 天内按剩余价值退。有朋友用了快一个月觉得不合适，退回了一部分，不是嘴上说说的政策。

5. **双向流量计费。** 上行和下行都算流量，不是只算下载。Malibu 1TB/月算双向，实际用起来大概是 500GB 纯下载的节奏，做节点的话够用。
